# gogenfilter

A Go library for detecting and filtering auto-generated code files. Designed for linters, static analysis tools, and code quality tools that need to skip generated output from tools like [sqlc](https://sqlc.dev), [templ](https://templ.guide), and [go-enum](https://github.com/abice/go-enum).

## Features

- **Multi-tool detection**: sqlc, templ, go-enum out of the box
- **Two-phase detection**: Filename-based first (zero I/O), then content-based (reads file)
- **Custom patterns**: Include/exclude patterns with wildcard support
- **Thread-safe metrics**: Track what was filtered and why
- **SQLC config discovery**: Parse `sqlc.yaml`/`sqlc.yml` to find output directories
- **Zero framework dependencies**: Only external dep is `github.com/go-faster/yaml` for SQLC config parsing

## Install

```bash
go get github.com/LarsArtmann/gogenfilter
```

## Quick Start

```go
package main

import (
    "fmt"
    "github.com/LarsArtmann/gogenfilter"
)

func main() {
    // Create a filter that catches all known generated code types
    f := gogenfilter.NewFilter(true, []gogenfilter.FilterOption{gogenfilter.FilterAll})

    // Check individual files
    if f.ShouldFilter("db/models.go") {
        fmt.Println("skipping generated file")
    }

    // Get stats about what was filtered
    stats := f.GetStats()
    fmt.Printf("Checked: %d, Filtered: %d\n", stats.TotalFilesChecked, stats.TotalFiltered())
}
```

## Detection Functions

Use low-level detection directly without the `Filter` struct:

```go
// Check by filename + content (both phases)
gogenfilter.IsSQLCGenerated("db/models.go", content)
gogenfilter.IsTemplGenerated("page_templ.go", content)
gogenfilter.IsGoEnumGenerated("status_enum.go", content)

// Check by filename only (zero I/O)
gogenfilter.MatchesSQLCFilename("db/query.sql.go") // true

// Check content patterns
gogenfilter.HasSQLCContent(content)
gogenfilter.HasSQLCCodePatterns(content)

// Combined detection (returns reason)
reason := gogenfilter.DetectGenerated("file.go", map[gogenfilter.FilterOption]bool{
    gogenfilter.FilterSQLC:   true,
    gogenfilter.FilterTempl:  true,
    gogenfilter.FilterGoEnum: true,
})
```

## Filter Options

```go
// Individual options
gogenfilter.NewFilter(true, []gogenfilter.FilterOption{
    gogenfilter.FilterSQLC,
    gogenfilter.FilterTempl,
    gogenfilter.FilterGoEnum,
})

// Or use FilterAll for everything
gogenfilter.NewFilter(true, []gogenfilter.FilterOption{gogenfilter.FilterAll})
```

## Include/Exclude Patterns

```go
f := gogenfilter.NewFilter(true, []gogenfilter.FilterOption{gogenfilter.FilterAll})

// Only analyze files matching these patterns
f.WithIncludePatterns([]string{"pkg/*", "internal/*.go"})

// Additionally exclude files matching these patterns
f.WithExcludePatterns([]string{"*.pb.go", "mocks/*"})
```

## SQLC Config Discovery

Find sqlc output directories from config files:

```go
// Find all sqlc.yaml/sqlc.yml files
configs, err := gogenfilter.FindSQLCConfigs([]string{"./..."})

// Parse a specific config
config, err := gogenfilter.ParseSQLCConfig("sqlc.yaml")

// Get output directories directly
dirs, err := gogenfilter.GetSQLOutputDirs([]string{"."})
```

## Metrics

Thread-safe metrics tracking is built in:

```go
f := gogenfilter.NewFilter(true, []gogenfilter.FilterOption{gogenfilter.FilterAll})

f.ShouldFilter("db/models.go")          // sqlc file
f.ShouldFilter("page_templ.go")         // templ file
f.ShouldFilter("main.go")               // regular file

stats := f.GetStats()
// stats.TotalFilesChecked == 3
// stats.FilteredByReason[gogenfilter.ReasonSQLC] == 1
// stats.FilteredByReason[gogenfilter.ReasonTempl] == 1
// stats.TotalFiltered() == 2
```

## Filter Reasons

| Constant | Meaning |
|---|---|
| `ReasonSQLC` | File generated by sqlc |
| `ReasonTempl` | File generated by templ |
| `ReasonGoEnum` | File generated by go-enum |
| `ReasonIncludePattern` | File excluded (didn't match include pattern) |
| `ReasonExcludePattern` | File excluded (matched exclude pattern) |
| `ReasonNotFiltered` | File was not filtered |
