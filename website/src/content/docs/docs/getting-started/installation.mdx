## Requirements

- Go 1.26 or later

## Install

```bash
go get github.com/LarsArtmann/gogenfilter
```

## Verify

```go
package main

import (
    "fmt"
    _ "github.com/LarsArtmann/gogenfilter"
)

func main() {
    fmt.Println("gogenfilter installed successfully")
}
```
