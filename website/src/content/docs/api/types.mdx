## FilterOption

A string type representing a generator category to filter.

```go
type FilterOption string
```

### Methods

- `String() string` — returns the underlying string value
- `Reason() (FilterReason, bool)` — returns the corresponding `FilterReason`. Returns `("", false)` for meta-options like `FilterAll` or unregistered options.
- `IsValid() bool` — reports whether the value is recognized

### Constants

`FilterSQLC`, `FilterTempl`, `FilterGoEnum`, `FilterProtobuf`, `FilterOapi`, `FilterDeepcopy`, `FilterWire`, `FilterMoq`, `FilterMockgen`, `FilterStringer`, `FilterMockery`, `FilterEnt`, `FilterGqlgen`, `FilterEasyjson`, `FilterMsgp`, `FilterCounterfeiter`, `FilterGoSwagger`, `FilterGeneric`, `FilterAll`

## FilterReason

A string type representing why a file was filtered.

```go
type FilterReason string
```

### Methods

- `String() string` — returns the underlying string value
- `IsValid() bool` — reports whether the value is recognized

### Constants

`ReasonSQLC`, `ReasonTempl`, `ReasonGoEnum`, `ReasonProtobuf`, `ReasonOapi`, `ReasonDeepcopy`, `ReasonWire`, `ReasonMoq`, `ReasonMockgen`, `ReasonStringer`, `ReasonMockery`, `ReasonEnt`, `ReasonGqlgen`, `ReasonEasyjson`, `ReasonMsgp`, `ReasonCounterfeiter`, `ReasonGoSwagger`, `ReasonGeneric`, `ReasonOutsideScope`, `ReasonExcludePattern`, `ReasonNotFiltered`

## FilterResult

Detailed result from `FilterDetailed` and `FilterDetailedWithContent`.

```go
type FilterResult struct {
    Filtered bool
    Reason   FilterReason
    Path     string
    Trace    string
}
```

### Methods

- `Is(reason FilterReason) bool` — reports whether this result was filtered with the given reason. Returns false if not filtered or filtered for a different reason.
- `String() string` — human-readable representation

```go
result, _ := f.FilterDetailed("db/models.go")
if result.Is(gogenfilter.ReasonSQLC) {
    fmt.Println("sqlc file:", result.Path)
}
```

## Derived Lists

### `AllFilterOptions() []FilterOption`

Returns all 19 valid `FilterOption` values (18 detector options + `FilterAll`).

### `AllGeneratorOptions() []FilterOption`

Returns all 18 detector `FilterOption` values (excludes `FilterAll`). Use when enumerating available generators rather than validating options.

### `AllFilterReasons() []FilterReason`

Returns all 21 valid `FilterReason` values.

All three are derived from the internal `detectors` table — `AllFilterOptions()` also includes `FilterAll`, and `AllFilterReasons()` also includes `ReasonOutsideScope`, `ReasonExcludePattern`, and `ReasonNotFiltered`.

## FilterConfig

A functional option type for configuring a `Filter` via `NewFilter`.

```go
type FilterConfig func(*Filter) error
```

Created by `WithFilterOptions`, `WithFS`, `WithIncludePatterns`, and `WithExcludePatterns`. Passed to `NewFilter(configs ...FilterConfig)`.

## Invariant

Each `FilterOption` and its matching `FilterReason` share the same underlying string value (e.g., `FilterSQLC == "sqlc" == ReasonSQLC`).
